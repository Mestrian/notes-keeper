# 📝 Notes App

Um site desenvolvido para criação, organização e gerenciamento de notas de forma simples, tendo todas as notas
salvas localmente.

# 🎯 Objetivos do Projeto

Os principais objetivos do projeto são:

* Criar um sistema simples de gerenciamento de notas
* Permitir organização e busca eficiente de informações
* Servir como projeto de estudo e prática de desenvolvimento
* Evoluir o sistema com novos recursos ao longo do tempo

# 🚀 Funcionalidades já Implementadas

Atualmente o projeto possui as seguintes funcionalidades:

* Criar novas notas
* Editar notas existentes
* Excluir notas
* Visualizar lista de notas
* Salvar notas localmente
* Adicionar tags as notas
