# 📝 Notes App

Um site desenvolvido para criação, organização e gerenciamento de notas de forma simples e eficiente.
O objetivo do projeto é fornecer um ambiente rápido para registrar ideias, tarefas e informações importantes. 
Nesse momento, todas as notas são salvas apenas na máquina do usuário, ou seja, totalmente local.

# 🎯 Objetivos do Projeto

Os principais objetivos do projeto são:

* Criar um sistema simples de gerenciamento de notas
* Permitir organização e busca eficiente de informações
* Servir como projeto de estudo e prática de desenvolvimento
* Evoluir o sistema com novos recursos ao longo do tempo

# 🚀 Funcionalidades já Implementadas

Atualmente o projeto possui as seguintes funcionalidades:

* Criar novas notas
* Editar notas existentes
* Excluir notas
* Visualizar lista de notas
* Salvar notas localmente
* Interface simples e rápida

# 🛠 Tecnologias Utilizadas

Este projeto utiliza as seguintes tecnologias:

* Linguagem: JavaScript 
* Framework: React, Tailwind
* Bibliotecas: CodeMirror 

# ⚙️ Como Executar o Projeto

## 1 - Clonar o repositório

```
git clone https://github.com/seu-usuario/notes-app.git
```

## 2 - Entrar na pasta

```
cd notes-app
```

## 3 - Instalar dependências

```
npm install
```

## 4 - Rodar o projeto

```
npm run dev
```
